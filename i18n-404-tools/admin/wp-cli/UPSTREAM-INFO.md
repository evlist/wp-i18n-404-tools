# Upstream Synchronization Metadata

This directory contains metadata for tracking synchronization with the upstream [wp-cli/i18n-command](https://github.com/wp-cli/i18n-command) project.

- `upstream-info.json`: Contains information about the last sync (upstream URL, commit, date, etc.).

**Note:** This file is for traceability and REUSE compliance only. It is not used by Composer or by the plugin code itself.
